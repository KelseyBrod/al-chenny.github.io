What is HTML?

HTML, also known as Hypertext Markup Language, is a front-end language that allows for text and information to be processed by a web browser. Not only do HTML elements allow for things like video, images, and links, but many also give semantic meaning.

Although we may never see the semantic meaning this can be helpful when styling the website later down the line.

What is CSS?

CSS, officially known as Cascading Style Sheets, is a language used in conjunction with HTML to add "styling" to a webpage. This tells the browser how to display all the text that was coded in html

aka it makes all the text pretty by adding colors, fonts, and changing the page the layout
