HTML Vocab

Elements

an element is represented by lil signs <> to indicate the start, content, and end of some piece of info

most elements will have a both an opening and closing tag, so <> and then after the end

Text to Audio

There are many elements that dont bring visual meaning, but with text to audio activated will give semantic meaning

Accessibility

Keep accessibility in mind! Make text easy to read, add visual cues with links, hovers etc.

Basic Markup Elements

You should probably use all of these

!DOCTYPE html
yeah just use it to indicate in visual studio code that this is indeed html
html
encase all elements inside the html element
head
contains meta tags, and other information for the website
meta
adds meta tags which can be helpful for SEO. common meta tags:
  • meta charset="UTF-8" lang = "en" dir="ltr"
  • meta name="author" content ="Allison Chen"
  • meta name="viewport" content="width=device-width, initial-scale=1"
body
encase any text of any kind in the body tag
header
text that displays at the top of a page
main
where the main text goes
footer
text that displays at the bottom of the page
p
p stands for paragraph. for the bulk of written information use this!
h1,h2,h3,h4,h5
these are headers, which slowly descend in size.

h1 being the largest,

and
h5 being the smallest
