CSS Basics

here are some common ways to style a website

element selector

element.{

put your stuff: in here;

}

linking a stylesheet

standard practice is to link a style sheet containing all the css tags instead of clogging up the html file with style elements

link href="../styles/style.css" type="text/css" rel="stylesheet"

need more help?

although honestly to search for specifics–– when in doubt–– w3schools.com

elements{styles}

body
  • background-color, background-image
  • text-align: center; left; right; justify;
img
  • border
  • border-radius
  • width/height
sizing
  • % - size is determined by the size of the screen. Good for responsive design!
  • fr - size is "fraction" of the screen. Good for responsive design!
  • px - size is set number of pixels
  • em - size is determined by size relative to the current font in use. Good for responsive design!
:pseudoclass

edits styles of the specific state of an element

  • :hover - when hovering over an element
  • :link - when a link hasn't been visited
  • :visited - when a link has been visited
transition

a common way to make an interactive websites is by using hover animations. A simple animation can be made with a by using {transtion: #s} and changing styles such as:

transform, color, rotate etc.

classes and ids

insert class="" or id="" into the opening tag

.class

applied more broadly to several elements

#id

more specific styling that normally only applies to one specific element
